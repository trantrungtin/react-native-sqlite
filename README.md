# react-native-sqlite

SQLCipher Native Plugin for React Native

## Installation

```sh
npm install react-native-sqlite
```

## Usage

```js
import Sqlite from "react-native-sqlite";

// ...

const result = await Sqlite.multiply(3, 7);
```
